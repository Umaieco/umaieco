## 🚀 Шаг 1: Подготовка (10 минут)

### 1.1 Установите Node.js
Скачайте и установите с https://nodejs.org (версия 18+)

### 1.2 Установите Git
Скачайте с https://git-scm.com

### 1.3 Проверьте установку
Откройте терминал (Командная строка / Terminal) и выполните:
```bash
node --version    # должно показать v18+ 
git --version     # должно показать git version ...
```

---

## 🚀 Шаг 2: Загрузка на GitHub (10 минут)

### 2.1 Создайте аккаунт на GitHub
Зайдите на https://github.com и зарегистрируйтесь (если ещё нет аккаунта).

### 2.2 Создайте новый репозиторий
1. Нажмите **"+"** → **"New repository"**
2. Название: `umai-eco`
3. Оставьте **Public** 
4. НЕ ставьте галочку "Add README"
5. Нажмите **"Create repository"**

### 2.3 Загрузите файлы проекта
Откройте терминал в папке проекта `umai-eco-vercel` и выполните:

```bash
cd путь/к/папке/umai-eco-vercel

git init
git add .
git commit -m "Initial commit: UMAI ECO website"
git branch -M main
git remote add origin https://github.com/ВАШ_ЛОГИН/umai-eco.git
git push -u origin main
```

Замените `ВАШ_ЛОГИН` на ваш логин GitHub.

---

## 🚀 Шаг 3: Деплой на Vercel (5 минут)

### 3.1 Создайте аккаунт на Vercel
1. Зайдите на https://vercel.com
2. Нажмите **"Sign Up"**
3. Выберите **"Continue with GitHub"** (войдите через GitHub)

### 3.2 Подключите проект
1. На главной странице Vercel нажмите **"Add New Project"**
2. Найдите репозиторий `umai-eco` и нажмите **"Import"**
3. Vercel автоматически определит Vite + React
4. Нажмите **"Deploy"**
5. Через 1-2 минуты сайт будет доступен по ссылке вида: `umai-eco.vercel.app`

---

## 🚀 Шаг 4: Регистрация домена umaieco.kg (1-2 дня)

### 4.1 Выберите регистратора
Домены .kg регистрируются через аккредитованных регистраторов:

| Регистратор | Сайт | Примерная цена |
|-------------|------|----------------|
| AsiaInfo | https://domain.kg | ~1000-1500 сом/год |
| Totel | https://www.totel.kg | ~1000 сом/год |
| WebHost | https://webhost.kg | ~1200 сом/год |

### 4.2 Зарегистрируйте домен
1. Зайдите на сайт регистратора
2. Проверьте доступность `umaieco.kg`
3. Оформите регистрацию (потребуется паспорт / данные компании)
4. Оплатите

---

## 🚀 Шаг 5: Подключение домена к Vercel (15 минут)

### 5.1 Добавьте домен в Vercel
1. Зайдите в ваш проект на Vercel
2. Перейдите в **Settings** → **Domains**
3. Введите `umaieco.kg` и нажмите **"Add"**
4. Vercel покажет DNS-записи, которые нужно добавить

### 5.2 Настройте DNS у регистратора
Зайдите в панель управления DNS вашего регистратора и добавьте записи:

```
Тип: A
Имя: @
Значение: 76.76.21.21

Тип: CNAME  
Имя: www
Значение: cname.vercel-dns.com
```

### 5.3 Дождитесь обновления DNS
DNS-записи обновляются от 10 минут до 48 часов (обычно 1-2 часа).

### 5.4 SSL-сертификат
Vercel **автоматически** выпустит бесплатный SSL-сертификат.
После этого сайт будет доступен по `https://umaieco.kg`
